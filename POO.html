1. De acordo com a apresentação, qual é o foco principal da Programação Orientada a Objetos (POO)?

Racional: A estruturação do programa é baseada nos dados, partindo do princípio que as funções mudam mais do que os dados.

2. O que a combinação de 'comportamento' e 'características' define em um contexto de Orientação a Objetos?

Racional: Um objeto é a junção de comportamento (métodos) e características (atributos), mantendo o estado encapsulado.

3. Qual é a definição de 'Classe' apresentada no contexto da Orientação a Objetos?

Racional: Uma classe serve como um molde ou categoria para agrupar objetos que compartilham as mesmas características (atributos) e comportamentos (métodos).

4. Qual dos seguintes não é um elemento listado na apresentação como parte do corpo de uma classe Java?

Racional: Objetos são instâncias da classe, criados a partir dela, mas não são declarados dentro do corpo da classe, ao contrário de atributos, métodos e construtores.

5. Na estrutura mínima de um programa Java, qual é o método por onde se inicia a execução, de acordo com o slide?

Racional: O método main (public static void main) é o ponto de entrada por onde se inicia a execução do programa Java.

6. Qual é a função da palavra reservada private ao declarar um atributo em Java?

Racional: O modificador private restringe o acesso aos atributos, garantindo o encapsulamento, o que é uma pré-condição para POO.

7. Em Java, o tipo String é classificado como:

Racional: Classes, como String, são classificadas como tipos de referência, e seus elementos são referências para objetos.

8. Qual é a distinção fundamental entre os elementos de um tipo primitivo (int, double) e os elementos de um tipo referência (classes, String) em Java?

Racional: Tipos primitivos armazenam o valor diretamente, enquanto tipos referência armazenam um endereço de memória que aponta para o objeto.

9. A expressão em Java: "maio " + " de " + 99 resulta em “maio de 99”. Qual recurso ou conceito Java é implicitamente demonstrado nesta operação?

Racional: O operador + de concatenação de strings executa automaticamente a conversão do inteiro 99 para o formato de string.

10. A declaração de atributos private String nome, sobrenome; é válida em Java, de acordo com a apresentação, e demonstra que:

Racional: A sintaxe permite economizar linhas de código agrupando atributos do mesmo tipo em uma única declaração, separados por vírgula.

11. Qual é o principal objetivo de um Método em uma classe Java, como o creditar(double valor)?

Racional: Um método representa o comportamento do objeto, e sua execução tipicamente manipula ou altera o estado do objeto por meio de seus atributos.

12. O que a palavra reservada void na assinatura de um método indica?

Racional: O void é o tipo de retorno para indicar que a função do método é realizar uma operação (efeito colateral) sem produzir um valor de saída.

13. Qual comando é utilizado dentro de um método, como o getNumero(), para enviar um valor de volta como resultado de sua execução?

Racional: O comando return é essencial para que métodos com um tipo de retorno diferente de void possam finalizar a execução e fornecer um valor ao chamador.

14. No método public void setNumero(String numero) { this.numero = numero; }, qual é a função do prefixo this. na expressão this.numero = numero?

Racional: Quando um parâmetro tem o mesmo nome de um atributo, this. é usado para garantir que o lado esquerdo (o atributo) seja referenciado corretamente.

15. A tabela 'Saldo', 'Número' muda de '875,32', '21.342-7' para '895,32', '21.342-7' após a execução do método Creditar(20). Isso ilustra o conceito de:

Racional: O estado do objeto, definido pelos valores de seus atributos (saldo), é alterado pela execução de um de seus métodos (Creditar).

16. Na POO, como a comunicação entre objetos é realizada para que eles executem tarefas em conjunto?

Racional: Chamar um método (por exemplo, conta.creditar()) é o mecanismo pelo qual um objeto envia uma 'mensagem' (requisição) para que outro execute uma operação.

17. Na expressão conta.creditar(45.30), o que representa a parte creditar?

Racional: O nome do método, seguido pelos parênteses e argumentos, define a operação a ser executada pelo objeto conta.

18. De acordo com o slide 'Imprimindo na tela', qual objeto especial em Java representa a tela do computador para a saída de informações?

Racional: A variável System.out contém a referência para o objeto que gerencia a saída de dados, como o console.

19. Qual regra de sintaxe fundamental se aplica à maioria dos comandos em Java, de acordo com as 'Regras para o Uso de Comandos'?

Racional: O ponto-e-vírgula é um terminador de declaração e comando fundamental na sintaxe Java, essencial para a maioria das instruções.

20. Qual é a regra importante de sintaxe e nomenclatura em Java mencionada, que se aplica a Nomes de Classes, Variáveis e Métodos?

Racional: Java é uma linguagem case-sensitive, o que significa que o uso de maiúsculas e minúsculas é levado em conta e distingue um nome do outro (por exemplo, Nome é diferente de nome).

21. Em Java, qual dos seguintes não é um tipo primitivo?

Racional: String é uma classe em Java, não um tipo primitivo. Os tipos primitivos incluem int, boolean, char, byte, short, long, float e double.

22. Qual é a finalidade de uma variável em Java?

Racional: Variáveis são usadas para armazenar dados, que são chamados de valores.

23. Qual das seguintes opções é uma regra válida para identificadores Java?

Racional: Identificadores não podem ser palavras reservadas (keywords) e não podem começar com um dígito. Eles podem ser arbitrariamente grandes e não podem conter espaços.

24. Qual é o tipo inteiro mais comum usado em Java, de acordo com os documentos?

Racional: O tipo `int` é o tipo inteiro mais comum mencionado nos documentos.

25. Qual é a sintaxe correta para declarar uma constante em Java?

Racional: A sintaxe para declarar uma constante é `[public] [static] final = ;`. O exemplo dado é `public static final double PI = 3.14159;`.

26. Para ler dados inseridos através do teclado em Java, qual classe é tipicamente utilizada?

Racional: A classe `Scanner` é usada para ler valores inseridos através do teclado, e requer a importação `import java.util.*;`.

27. Qual operador aritmético binário é usado para obter o resto da divisão inteira?

Racional: O operador `%` (resto) é usado com operandos inteiros para obter o resto da divisão inteira.

28. Qual é o resultado da expressão `99 / 100` em Java, quando ambos os operandos são inteiros?

Racional: Quando ambos os operandos da divisão são inteiros, o resultado é a divisão inteira, que trunca o valor. Assim, `99/100` resulta em `0`.

29. Qual operador lógico representa a operação "OU" (OR)?

Racional: O operador `||` representa a operação lógica OU (OR).

30. Em Java, qual é a precedência mais alta entre os operadores listados?

Racional: Os operadores unários como `++` e `--` têm a precedência mais alta, seguidos pelos operadores aritméticos binários de multiplicação/divisão/resto, e depois pelos de adição/subtração.

31. Qual será o valor de `result` e `m` após a execução do seguinte código? 

int m = 4;
int result = 3 * (++m);

Racional: O operador `++m` (pré-incremento) incrementa `m` para 5 ANTES de usar seu valor na expressão. Então, `result = 3 * 5 = 15`.

32. Qual será o valor de `result` e `m` após a execução do seguinte código? 

int m = 4;
int result = 3 * (m++);

Racional: O operador `m++` (pós-incremento) usa o valor atual de `m` (4) na expressão ANTES de incrementá-lo. Então, `result = 3 * 4 = 12`, e depois `m` se torna 5.

33. Qual é o índice do caractere 'J' na string "Java é divertido."?

Racional: As posições em uma string (índices) começam em 0. Portanto, o primeiro caractere 'J' está no índice 0.

34. Qual tipo de comentário em Java pode ser extraído automaticamente por software Java (como o Javadoc)?

Racional: Comentários Javadoc, que começam com `/**` e terminam com `*/`, são projetados para serem extraídos automaticamente para gerar documentação.

35. Qual é a convenção de nomenclatura recomendada para nomes de variáveis em Java, de acordo com os documentos?

Racional: A convenção recomendada é começar variáveis com letras minúsculas e usar letras maiúsculas para separar palavras (camelCase), como `corDeFundo`.

36. O que significa dizer que um programa é "auto-documentado"?

Racional: Programas auto-documentados são aqueles que possuem um estilo claro, nomes bem escolhidos para variáveis e métodos, e são bem comentados para explicar seu funcionamento.

37. Qual é o propósito principal do encapsulamento em Programação Orientada a Objetos, conforme sugerido pelo uso da palavra-chave `private`?

Racional: O encapsulamento, implementado com `private`, garante que os atributos de um objeto só possam ser acessados e modificados através dos métodos definidos na própria classe, protegendo a integridade dos dados.

38. Em Java, quando os operadores binários têm igual precedência, qual é a regra de avaliação?

Racional: Quando os operadores binários têm igual precedência, a avaliação é feita da esquerda para a direita (associatividade).

39. Qual é a principal diferença entre um objeto e uma classe?

Racional: Uma classe é uma abstração ou um "projeto" para criar objetos, enquanto um objeto é uma entidade real e concreta criada a partir de uma classe, com seus próprios valores para os atributos.

40. O que acontece quando um dos operandos em uma operação aritmética é um tipo em vírgula flutuante e o outro é um inteiro?

Racional: Quando há uma mistura de tipos em uma operação aritmética (um flutuante e um inteiro), o resultado é promovido para o tipo mais abrangente, que neste caso é o tipo em vírgula flutuante.
